code

My first coding experience came from adding HTML tags and embedding videos into the articles I wrote at my college newspaper. After spending a few years working as a writer and website manager, however, I realized not only just how much getting my work done depended on having good software at my disposal, but also how little I really knew about the systems I was working with every day. In 2019, I decided to change that and enrolled in a Java Bootcamp at Grand Circus in Detroit. There, I added the following languages and skills to my repertoire:

    Java Spring MVC SQL
    Git/GitHub Hibernate HTML/CSS
    OOP Web APIs Adobe Illustrator / InDesign

You'll find info on some of the programs I've made below, but you can find even more of my work on my GitHub page.

  • minefield

    This Minesweeper port was a lot of fun to make. Working with a partner, I first made a version of the game that used a .txt file to store and display the minefield and took user input for which square to uncover using a basic Java compiler console. Once we had the basic logic down, I spent the weekend researching JSwing and created the GUI you can see in the accompanying gif. It includes a soundtrack of music I recorded at my home studio and a pretty realistic explosion sound for when the player loses.
