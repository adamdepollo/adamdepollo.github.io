SOFTWARE

My first coding experience came from adding HTML tags and embedding videos into the articles I wrote at my college newspaper. After spending a few years working as a writer and website manager, however, I realized not only just how much getting my work done depended on having good software at my disposal, but also how little I really knew about the systems I was working with every day. In 2019, I decided to change that and enrolled in a Java Bootcamp at Grand Circus in Detroit. There, I added the following languages and skills to my repertoire:

Spring MVC

SQL (MySQL)
